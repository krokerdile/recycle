## 소개
마크다운 에디터와 라인코멘트를 통한 가장 쉬운 코드리뷰 공간을 제공합니다

## 기술 스택

- FE
    - React, TypeScript
    - ReactQuery
    - Zustand
    - React-md-editor
- BE
    - Spring Boot
    - MySQL
    - Rabbit MQ
    - Redis
    - Elasticsearch
    - Logstash
    - Kibana
    - Flyway
